B - Initialization

In this tutorial, you will be building a scene similar to the one above.


  • 1. To begin, click New Project and Browse to find an empty folder to store this project in. Any empty folder will work.

  • 2. Click Create & Edit to create and launch a blank project.

  • 3. In the Scene Tab on the left-hand side of the screen, click 3D Scene to create our first Node. A Node is the term for any object added to a scene.

  • 4. When you clicked 3D Scene, Godot created a Spatial node that will be the Parent of all your nodes created from this point onward. Right click the newly added Spatial node and Rename it to whatever you'd like. I named mine "World" to signify its function to our scene.

  • 5. This tab above your scene view is telling you that we haven't saved this scene yet. To save, push ctrl+S(windows)/cmd+S(mac) OR click Scene and then Save As in the top-left of your screen.

Footnote I: Purple text will be used to highlight key terms that are worth remembering. Not all terminology is crutial for this lesson, so don't be too worried about remembering unhighlighted terms (such as Spatial on this page).
