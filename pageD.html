D - Creating Child Nodes

  • 1. With the World node selected, click the + button to create a child node. This button will always create a child of the selected node, so it's very important to have the correct node selected before pressing it.

  • 2. Search "static" to find the StaticBody node. Double click StaticBody to add this node to your scene. The StaticBody is intended to be a physics object that doesn't move, so it'll be perfect for the floor of the scene.

  • 3. As you may notice, you have a yellow caution symbol next to your new child. For our physics "body" nodes, they will require a CollisionShape to be able to interact or do anything to do with physics in this scene.

  • 4. Just as you did in step 2 of this page, add a child (+ button with StaticBody selected) to the StaticBody. This time, search "collision" and double click CollisionShape. For clarity sake, rename the StaticBody to "Floor".

    Remember, to rename a node right click it in the list of nodes in our scene and select Rename.

Footnote I: All nodes you create for this tutorial will be 3D, which is marked by pale red icons next to the node names. Blue, green, and white nodes should effectively be ignored.
